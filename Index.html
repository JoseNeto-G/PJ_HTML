O que é HTML?

Voltar ao topo

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ele estrutura o conteúdo na internet, permitindo que navegadores interpretem e exibam textos, imagens, vídeos e outros elementos.

Estrutura Básica do HTML

Voltar ao topo

Um documento HTML básico é composto por várias seções principais, incluindo a declaração do tipo de documento, a seção head (cabeçalho) e a seção body (corpo).

Tags

Voltar ao topo

Tags são elementos que definem a estrutura e o conteúdo de uma página HTML. Cada tag é composta por um nome entre chaves angulares, como <h1> para títulos e <p> para parágrafos.

Atributos Básicos das Tags

Voltar ao topo

Atributos são propriedades que podem ser adicionadas às tags para modificar seu comportamento ou aparência. Exemplos comuns incluem o atributo "class" e "id", que ajudam a identificar elementos no CSS e JavaScript.

Listas Ordenadas e não ordenadas

Voltar ao topo

Listas em HTML podem ser ordenadas (com números) ou não ordenadas (com marcadores). As listas ordenadas usam a tag <ol>, enquanto as não ordenadas usam <ul>. Cada item da lista é representado pela tag <li>.

Links

Voltar ao topo

Links em HTML são criados com a tag <a>. O atributo href especifica o endereço do link, permitindo navegar entre páginas ou recursos externos.
